Vectorworks Stream Deck Profile

28 april 2020 · Sebastiaan ten Broek

Veel mensen in de AV sector maken gebruik van een Elgato Stream Deck. Dit is een programmeerbare USB keypad met 6, 15 of 32 knoppen die ieder van een klein LCD schermpje zijn voorzien. Iedere knop kan door de gebruiker van een zelf ingestelde functie met eigen achtergrond afbeelding worden voorzien. Download onderaan dit artikel mijn Vectorworks Stream Deck Profile.

Je kunt bijvoorbeeld een toetsencombinatie onder 1 knop opslaan. Maar ook andere functies zoals opstarten van programma’s, bestandsmappen, website’s links je kunt het zo gek niet bedenken of je kunt het met 1 druk op de knop met je Stream Deck aansturen. Ontwikkelaars kunnen er zelf plugins voor schrijven zodat andere software en zelfs apparatuur er mee aangestuurd kan worden. Het is eigenlijk een soort algemene console voor bijna alles.

De Stream Deck was op de markt gebracht om online streamers hun streams met 1 handeling te laten starten en stoppen. Maar inmiddels is dit apparaat ook door de AV wereld omarmd. Software bedrijf Bitfocus schreef een plugin voor de Stream Deck die een interface vormt tussen de Stream Deck en een lange lijst van apparaten die op evenementen gebruikt worden. Denk daarbij aan video switchers, audio FOH desks, lichttafel de lijst is heel erg lang.

Vectorworks Profile

Zelf heb ik een mooi Vectorworks Stream Deck Profile aangemaakt voor gebruik met Vectorworks. Vrijwel alle tools en de meest gangbare menu commands hebben hierin een knop met het originele Vectorworks tool icoon. Zo kun je je muis bij het model houden en met je vrije hand de door jou gewenste Vectorworks functie oproepen. Je kan mijn profiel hieronder downloaden.

Let op dat dit profiel is gemaakt met een Mac. Dat betekend dat alle hotkeys zijn gemaakt met de CMD toets. Mocht je dit profiel op een Windows machine willen gebruiken dan moet je in de Stream Deck configuratie de hotkeys opnieuw intoetsen met CTRL. Ook kan het zijn dat een van mijn toetsen combinaties niet aanwezig is in jouw Vectorworks workspace. In Vectorworks worden de toetsencombinaties namelijk opgeslagen in je persoonlijke workspace. Dan zou je je eigen toetsencombinatie kunnen gebruiken, of je workspace aanpassen zodat mijn toetsencombinatie ook in jouw workspace voorkomt.
